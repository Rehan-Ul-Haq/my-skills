# Section Patterns by Project Type

## CLI Application

```markdown
# Name
Description

## Installation
## Usage
## Commands
## Examples
## Configuration (if applicable)
## License
```

## Library/Package

```markdown
# Name
Description + badges

## Installation
## Quick Start
## API Reference (or link)
## Examples
## Contributing
## License
```

## Web Application

```markdown
# Name
Description + screenshot

## Features
## Installation
## Configuration
## Usage
## Deployment
## License
```

## API Service

```markdown
# Name
Description

## Getting Started
## Authentication
## Endpoints (summary)
## Examples
## Error Handling
## License
```

---

## Section Details

### Title
- Match package/repo name
- No redundant "README" text

### Description
- One sentence, what it does
- Optional: badges after title

### Installation
- Exact commands
- Prerequisites listed first
- Platform variations if needed

### Usage
- Most common use case first
- Code blocks with language tags
- Expected output shown

### Examples
- Real-world scenarios
- Copy-pasteable code
- Increasing complexity
