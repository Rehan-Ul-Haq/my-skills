# skill-creator-enhanced

[![Version](https://img.shields.io/badge/Version-1.0.0-blue)](CHANGELOG.md)

Production-grade skill creation guide that scores 96/100 on skill-validator.

## Why Use This?

| Problem | Solution |
|---------|----------|
| Skills with wrong assumptions | Required Clarifications pattern |
| Missing error handling | Technical patterns reference |
| No quality consistency | Output Checklist + Must Follow/Avoid |
| Bloated SKILL.md files | Progressive disclosure patterns |

## Key Features

- **Clarification Questions** - Ask before building, prevent rework
- **Scope Clarity** - "Does / Does NOT do" sections
- **Enforcement Checklists** - Must Follow / Must Avoid patterns
- **Output Checklist** - Quality gate before delivery
- **3 Skill Types** - Builder, Guide, Automation patterns
- **5 Reference Files** - Deep guidance loaded only when needed

## Quick Start

```
Create a skill for [your domain]
```

The skill will:
1. Ask clarifying questions (skill type, domain, examples)
2. Guide through 7-step creation process
3. Ensure all quality criteria are met

## Structure

```
skill-creator-enhanced/
├── SKILL.md (265 lines)
└── references/
    ├── skill-patterns.md      # SKILL.md structure
    ├── quality-patterns.md    # Clarifications, enforcement
    ├── technical-patterns.md  # Errors, security, deps
    ├── workflows.md           # Workflow patterns
    └── output-patterns.md     # Templates, examples
```
